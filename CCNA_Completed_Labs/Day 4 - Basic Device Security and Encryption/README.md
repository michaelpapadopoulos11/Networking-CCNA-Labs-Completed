# Day 4 - Basic Device Security Lab and Encryption

A short introductory lab focused on securing Cisco IOS devices. I learned the difference between the CLI **enable password** and the **enable secret** commands, how each is stored, and why the secret always overrides the password. The lab also covered basic hostname configuration, enabling password encryption, and saving device configurations.
