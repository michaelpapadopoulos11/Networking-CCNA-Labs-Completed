## Day 11 (Part 1) – Configuring Static Routes

In this lab, I configured basic network settings across all devices, including hostnames, IP addresses, subnet masks, and default gateways. Once the addressing was in place, I added static routes on R1, R2, and R3 using the `ip route` command. These next‑hop routes allowed each router to learn networks they weren’t directly connected to, enabling full end‑to‑end connectivity between PC1 and PC2.
